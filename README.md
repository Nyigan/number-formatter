## Number Formatter
=========

A small library that adds commas to numbers

## Installation

  `npm install @towett/number-formatter`

## Usage

    var numFormatter = require('@towett/number-formatter');

    var formattedNum = numFormatter(35666);


  Output should be `35,654`


## Tests

  `npm test`
